# Secure Coding Review

## Project Overview
This project demonstrates a secure coding review of a Python-based login system. The project identifies common security vulnerabilities and provides a secure implementation using best coding practices.

## Features
- User Authentication
- Password Hashing using SHA-256
- Login Attempt Limitation
- Password Change Verification
- Input Validation
- Secure Coding Practices

## Technologies Used
- Python
- Jupyter Notebook
- GitHub
